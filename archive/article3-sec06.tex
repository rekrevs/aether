\section{Coupling Strength and Dimensional Analysis}

\textbf{Power form (dimension fixing):} the flow is written directly as power (using $\Psig$ for the scalar power, distinct from the 4-current $\Jsig$):
\begin{equation}
  \Psig(e) \;=\; \varepsilon\,\Ksig(e)\,\mathcal Q(e,t)\,
                 \tilde{\Delta\Phi}(\tilde e)\,P_{\rm pump}(e)
  \quad\text{(units: W)}.
  \tag{6.1}
\end{equation}
where $(e)$ denotes a single edge or calibrated pair, and:
\begin{itemize}
\item $\varepsilon$ dimensionless coupling strength,
\item $\Ksig(e)=\exp[-d_\sigma/\lambda_\sigma]$ dimensionless similarity factor for edge $(e)$,
\item $\mathcal Q(e,t)\in[0,1]$ dimensionless pattern quality,
\item $\tilde{\Delta\Phi}(\tilde e)$ dimensionless (normalized free energy),
\item $P_{\rm pump}(e)$ has unit W.
\end{itemize}

\textbf{Unit analysis:} $[\Psig]=\mathrm{W}$.

\textbf{Alternative rate form:} Define pump rate $\PumpRate:=P_{\rm pump}/(\hbar\omega_0)\,[\mathrm{s^{-1}}]$ to write
\begin{equation}
\Psig = \varepsilon\,\hbar\omega_0\,\Ksig\,\mathcal Q\,\PumpRate\,\tilde{\Delta\Phi}.
\tag{6.2}
\end{equation}

\textbf{Degeneracy dilution.} For a periodic system with $(N)$ equivalent matches, $(\Psig \to \Psig / N)$. For a cubic lattice with volume V and lattice constant a: $(N \sim V/a^3 \to \infty)$, explaining absence in homogeneous systems.
